Goal
In this last step of the cube, the ultimate goal is to complete the entire the cube ( in technical terms, this is called PLL, a shorthand for permute last layer. ).
Complete the headlights

The first substep is to complete the corners or make all four sides have “headlights”. In order to do so, you have to first find “headlights” of the cube that looks something like this:

It does not matter which color is in the middle or which color the headlight is, it just has to form a headlight, with the colors of two corners facing towards you having the same color.

Depending on the number of headlights, you can end up in different cases:

Case 1). 0 Headlight (no headlights on all four sides)
Case 2). 1 Headlight (headlight on one side only)
Case 3). 4 Headlights (headlights on all four sides)
=> You are lucky, you can skip this substep.
Complete the edges/rest of the cube
The final substep is to complete the rest of the cube. Once you have completed the headlights, there are only 4 cases you can encounter. For each of these 4 cases, there is a corresponding algorithm with which you can solve the cube. Examine the cube, then execute the corresponding algorithm.
Case 1). “Side stickers swapped”
Case 2). “Opposite stickers swapped”
Case 3). “1 bar completed + Clockwise swapped”
Case 4). “1 bar completed + Counter-clockwise swapped”

Congratulations. Now you have solved your first cube! While there seems to be a lot of algorithms to memorize, with enough practice you will be able to memorize them all. In fact, since this algorithm is based on the CFOP method, if you have mastered the concepts here you will be able to naturally progress towards the full CFOP method.
